# DDoS_Detection_ubuntu
DDoS Detection Within Ubuntu System


This Project Basically Predicts Probability of DDoS on Ubuntu Based on Following Features:
### pktcount,
### byteperflow,
### tot_kbps,
### rx_kbps,
### flows,
### bytecount,
### tot_dur,
### Protocol_ICMP,
### Protocol_TCP,
### Protocol_UDP,
### Protocol_HTTP,
### Protocol_HTTPS,
### Protocol_SSH,
### Protocol_DHCP,
### Protocol_FTP,
### Protocol_SMTP,
### Protocol_POP3,
### Protocol_IMAP,
### Protocol_DNS,
### src_ip,
### dst_ip
